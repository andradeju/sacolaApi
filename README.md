# sacolaApi
## Desenvolvimento de Sacola API para servir uma aplicação com domínio de delivery. 
### Projeto proposto no Bootcamp da Dio.

## Features
- Incluir itens na sacola
- Visualizar sacola
- Fechar sacola

## Ferramentas e Tecnologias:

- JAVA JDK
- GIT
- GRADLE
- POSTMAN
- Spring initialzr
- Spring boot starter web
- Lombok
- h2
